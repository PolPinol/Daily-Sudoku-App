# Sudoku App iOS
This is a simple Sudoku app for iOS. It is written in Flutter and uses the sudoku-api.vercel.app API to generate the puzzles.

## Requirements

* Xcode 8.0 or later
* iOS 10.0 or later
* Swift 3.0 or later

## Installation

1. Clone the repository
2. Open the project in Xcode
3. Build and run the project
